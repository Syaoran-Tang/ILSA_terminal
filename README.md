## ILSA
Source code for ILSA(An integrated isotopic labeling and freeze sampling apparatus) host program.

## Dependency
- VisualStudio 16.0 or higher
- C# .NetFramework v4.7.2 or higher
- HslCommunication [http://www.hslcommunication.cn/]
